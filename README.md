Business Sales Performance Analytics

Project Overview
This project analyzes business sales data using Power BI to identify sales trends, regional performance, product performance, and payment mode analysis.

Tools Used
- Power BI
- Microsoft Excel

Dataset Fields
- Order ID
- Customer ID
- Customer Name
- Order Date
- Product Name
- Category
- Region
- Quantity
- Sales
- Profit
- Payment Mode

Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Sales by Region
- Sales by Category
- Sales by Payment Mode
- Top Selling Products
- Region Filter (Slicer)

Key Insights
1. Total Sales: ₹405K
2. Total Profit: ₹78K
3. South region generated the highest sales.
4. Technology category contributed the highest revenue.
5. Cash was the most preferred payment mode.
